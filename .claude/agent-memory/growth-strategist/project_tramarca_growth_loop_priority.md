Loop #1 priorizado tras audit growth tramarca.es 2026-04-16:

**Flujo:** LinkedIn post semanal (Fernando) → Landing `/manual-demo` gated (email + rol) → entrega PDF + secuencia 5 emails (2/5/10/20/40 días) → CTA Founding Program email #5 → form contacto corto → propuesta + Stripe 24h → kickoff 48h → entrega → día +14 request testimonial + 2 referidos → referidos vuelven al loop.

**Why:** CAC ~20-40€ (solo tiempo), ciclo 14-60 días, usa activos ya producidos (manual Tramarca 34pp ya existe), funnel completo medible desde semana 2. Descartado Google Ads (sin PMF), SEO-pilar como motor (complemento, no loop), cold outbound (caro en setup).

**How to apply:**
- Priorizar este loop ANTES que cualquier otra actividad growth (SEO, ads, outbound) las próximas 8 semanas.
- No meter segundo canal hasta que este loop sea profitable y predictable.
- **Kill criteria:** 8 semanas con 0 Founding cerrados DESDE este loop específico (no contar referidos directos offline) → pivot a outbound LinkedIn 1-a-1 manual. No "darle más tiempo" — la señal es clara a las 8 semanas si el loop va a funcionar.

**Top 3 acciones semana 1 post-audit:**
1. Construir `/manual-demo` gated (~4h)
2. Acortar form contacto a 1 pantalla + step 2 post-submit por email (~2h)
3. Fix 404 `/nosotros` + reemplazar CTA home "Hablemos de tu marca" → "Ver propuesta en 24h" (~1h)

**Métrica review:** 4 semanas desde go-live. KPIs: downloads lead magnet, email→contact rate, form submit rate, lead→proposal→won rate.
